# Chart.js 圖表座標軸與資料需求表
以下是各種 Chart.js 圖表所需的座標軸資料與實例範例：

## 基本圖表類型
| 圖表類型 | 必要座標軸/資料 | 選用座標軸/資料 | 資料範例 | Chart.js 程式碼 |
|---------|--------------|--------------|---------|-------------|
| 折線圖 (Line Chart) | x軸: 類別/時間<br>y軸: 數值 | 多條線(datasets) | {labels: ['一月', '二月', '三月'], datasets: [{label: '銷售額', data: [65, 59, 80]}]} | ```const myChart = new Chart(ctx, {type: 'line', data: {labels: ['一月', '二月', '三月'], datasets: [{label: '銷售額', data: [65, 59, 80], borderColor: 'blue'}]}, options: {}});``` |
| 柱狀圖 (Bar Chart) | x軸: 類別<br>y軸: 數值 | 多組柱(datasets) | {labels: ['紅色', '藍色', '黃色'], datasets: [{label: '銷量', data: [12, 19, 3]}]} | ```const myChart = new Chart(ctx, {type: 'bar', data: {labels: ['紅色', '藍色', '黃色'], datasets: [{label: '銷量', data: [12, 19, 3], backgroundColor: ['red', 'blue', 'yellow']}]}, options: {}});``` |
| 雷達圖 (Radar Chart) | 極軸: 類別<br>徑向: 數值 | 多組資料(datasets) | {labels: ['攻擊', '防禦', '速度', '力量', '敏捷'], datasets: [{label: '角色A', data: [65, 59, 90, 81, 56]}]} | ```const myChart = new Chart(ctx, {type: 'radar', data: {labels: ['攻擊', '防禦', '速度', '力量', '敏捷'], datasets: [{label: '角色A', data: [65, 59, 90, 81, 56], fill: true, backgroundColor: 'rgba(54, 162, 235, 0.2)', borderColor: 'rgb(54, 162, 235)'}]}, options: {}});``` |
| 圓餅圖 (Pie Chart) | 扇形: 各部分數值 | 顏色 | {labels: ['紅色', '藍色', '黃色'], datasets: [{data: [300, 50, 100], backgroundColor: ['#FF6384', '#36A2EB', '#FFCE56']}]} | ```const myChart = new Chart(ctx, {type: 'pie', data: {labels: ['紅色', '藍色', '黃色'], datasets: [{data: [300, 50, 100], backgroundColor: ['#FF6384', '#36A2EB', '#FFCE56']}]}, options: {}});``` |
| 環形圖 (Doughnut Chart) | 扇形: 各部分數值 | 顏色 | {labels: ['紅色', '藍色', '黃色'], datasets: [{data: [300, 50, 100], backgroundColor: ['#FF6384', '#36A2EB', '#FFCE56']}]} | ```const myChart = new Chart(ctx, {type: 'doughnut', data: {labels: ['紅色', '藍色', '黃色'], datasets: [{data: [300, 50, 100], backgroundColor: ['#FF6384', '#36A2EB', '#FFCE56']}]}, options: {cutout: '70%'}});``` |
| 極座標圖 (Polar Area Chart) | 扇形: 各部分數值 | 顏色 | {labels: ['紅色', '藍色', '黃色'], datasets: [{data: [11, 16, 7], backgroundColor: ['#FF6384', '#36A2EB', '#FFCE56']}]} | ```const myChart = new Chart(ctx, {type: 'polarArea', data: {labels: ['紅色', '藍色', '黃色'], datasets: [{data: [11, 16, 7], backgroundColor: ['#FF6384', '#36A2EB', '#FFCE56']}]}, options: {}});``` |
| 氣泡圖 (Bubble Chart) | x軸: 數值<br>y軸: 數值<br>r軸: 泡泡大小 | 顏色 | {datasets: [{label: 'First Dataset', data: [{x: 20, y: 30, r: 15}, {x: 40, y: 10, r: 10}]}]} | ```const myChart = new Chart(ctx, {type: 'bubble', data: {datasets: [{label: 'First Dataset', data: [{x: 20, y: 30, r: 15}, {x: 40, y: 10, r: 10}], backgroundColor: 'rgba(255, 99, 132, 0.2)', borderColor: 'rgb(255, 99, 132)'}]}, options: {}});``` |
| 散點圖 (Scatter Chart) | x軸: 數值<br>y軸: 數值 | 多組點(datasets) | {datasets: [{label: 'Scatter Dataset', data: [{x: -10, y: 0}, {x: 0, y: 10}, {x: 10, y: 5}]}]} | ```const myChart = new Chart(ctx, {type: 'scatter', data: {datasets: [{label: 'Scatter Dataset', data: [{x: -10, y: 0}, {x: 0, y: 10}, {x: 10, y: 5}], backgroundColor: 'rgba(75, 192, 192, 0.2)', borderColor: 'rgb(75, 192, 192)'}]}, options: {}});``` |

## 混合圖表類型
| 圖表類型 | 必要座標軸/資料 | 選用座標軸/資料 | 資料範例 | Chart.js 程式碼 |
|---------|--------------|--------------|---------|-------------|
| 柱狀圖+折線圖 | x軸: 類別<br>y軸: 數值 | 多類型datasets | {labels: ['一月', '二月', '三月'], datasets: [{type: 'bar', label: '實際', data: [10, 20, 30]}, {type: 'line', label: '目標', data: [15, 25, 35]}]} | ```const myChart = new Chart(ctx, {type: 'bar', data: {labels: ['一月', '二月', '三月'], datasets: [{type: 'bar', label: '實際', data: [10, 20, 30]}, {type: 'line', label: '目標', data: [15, 25, 35]}]}, options: {}});``` |
| OHLC+成交量混合圖 | x軸: 時間<br>y軸1: 價格<br>y軸2: 成交量 | 雙y軸設定 | {datasets: [{type: 'ohlc', yAxisID: 'price', data: [{t: new Date(), o: 150, h: 155, l: 148, c: 153}]}, {type: 'bar', yAxisID: 'volume', data: [{t: new Date(), y: 10000}]}]} | ```const myChart = new Chart(ctx, {type: 'bar', data: {datasets: [{type: 'ohlc', yAxisID: 'price', data: [{t: new Date(), o: 150, h: 155, l: 148, c: 153}]}, {type: 'bar', yAxisID: 'volume', data: [{t: new Date(), y: 10000}]}]}, options: {scales: {yAxes: [{id: 'price', type: 'linear', position: 'left'}, {id: 'volume', type: 'linear', position: 'right'}]}}});``` |
| OHLC+移動平均線+KD線 | x軸: 時間<br>y軸1: 價格/MA<br>y軸2: KD值 | 多指標技術分析 | {datasets: [{type: 'ohlc', yAxisID: 'price', label: 'OHLC', data: [{t: new Date(), o: 150, h: 155, l: 148, c: 153}]}, {type: 'line', yAxisID: 'price', label: 'MA5', data: [{t: new Date(), y: 151}]}, {type: 'line', yAxisID: 'kd', label: 'K值', data: [{t: new Date(), y: 70}]}, {type: 'line', yAxisID: 'kd', label: 'D值', data: [{t: new Date(), y: 60}]}]} | ```const myChart = new Chart(ctx, {type: 'bar', data: {datasets: [{type: 'ohlc', yAxisID: 'price', label: 'OHLC', data: [{t: new Date(), o: 150, h: 155, l: 148, c: 153}]}, {type: 'line', yAxisID: 'price', label: 'MA5', data: [{t: new Date(), y: 151}]}, {type: 'line', yAxisID: 'kd', label: 'K值', data: [{t: new Date(), y: 70}]}, {type: 'line', yAxisID: 'kd', label: 'D值', data: [{t: new Date(), y: 60}]}]}, options: {scales: {yAxes: [{id: 'price', type: 'linear', position: 'left'}, {id: 'kd', type: 'linear', position: 'right', min: 0, max: 100}]}}});``` |
| 折線圖+氣泡圖 | x軸: 數值/時間<br>y軸: 數值 | 氣泡大小(r) | {datasets: [{type: 'line', data: [65, 59, 80]}, {type: 'bubble', data: [{x: 10, y: 60, r: 15}]}]} | ```const myChart = new Chart(ctx, {type: 'line', data: {datasets: [{type: 'line', data: [65, 59, 80]}, {type: 'bubble', data: [{x: 10, y: 60, r: 15}]}]}, options: {}});``` |
| 柱狀圖+散點圖 | x軸: 類別<br>y軸: 數值 | 多類型datasets | {labels: ['A', 'B', 'C'], datasets: [{type: 'bar', data: [10, 20, 30]}, {type: 'scatter', data: [{x: 0, y: 15}, {x: 1, y: 25}, {x: 2, y: 35}]}]} | ```const myChart = new Chart(ctx, {type: 'bar', data: {labels: ['A', 'B', 'C'], datasets: [{type: 'bar', data: [10, 20, 30]}, {type: 'scatter', data: [{x: 0, y: 15}, {x: 1, y: 25}, {x: 2, y: 35}]}]}, options: {}});``` |
